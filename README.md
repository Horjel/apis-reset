# Hello HTTP API

Primera API del proyecto de aprendizaje **APIS RESET**.

## Estado

Fase 1: proyecto base de Spring Boot creado con Java 17, Maven y Spring Web.

Todavia no contiene endpoints. El primer endpoint se implementara en la siguiente fase.

## Ejecutar localmente

```powershell
mvn spring-boot:run
```

La aplicacion escucha por defecto en `http://localhost:8080`.
